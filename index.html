2단계. Claude Code 설정

  1. Claude 데스크톱을 다운로드·설치한 뒤 로그인합니다.
  2. Claude 앱을 열고 상단의 Code 탭을 클릭합니다.
  3. 시작 화면에서 Local을 선택합니다. (내 컴퓨터 폴더에 직접 접근하는 방식)
  4. 작업할 폴더를 선택(새 폴더 생성)하고, 사용할 모델을 고릅니다. (Sonnet 4.6)
  5. Claude 앱 설정을 변경합니다. 앱 메뉴에서 설정을 열고 Claude Code를 클릭한 뒤:
    1. 권한 우회 모드 → ON
    2. 미리보기 → ON
    3. 미리보기 세션 유지 → ON
  6. 채팅창 아래에서 권한 건너뛰기를 켭니다. (파일 수정·터미널 명령을 자동 실행)

2단계. 터미널에서 설치

터미널 여는 법 ⌘ Cmd + Space → Spotlight 검색창에 터미널 → Return
  1. 1 Git

    설치 창이 열리면 '설치'를 누릅니다.

    xcode-select --install
  2. 2 Node.js LTS

    설치 창이 열립니다. 계속 '계속'을 눌러 완료하세요.

    PKG_URL=$(curl -fsSL https://nodejs.org/dist/index.json | ruby -rjson -e 'd=JSON.parse(STDIN.read); v=d.find{|x|x["lts"]}["version"]; puts "https://nodejs.org/dist/#{v}/node-#{v}.pkg"') curl -fsSL "$PKG_URL" -o /tmp/node-lts.pkg && open /tmp/node-lts.pkg
  3. 3 GitHub CLI

    설치 창이 열립니다. '계속'을 눌러 완료하세요.

    URL=$(curl -fsSL https://api.github.com/repos/cli/cli/releases/latest | ruby -rjson -e 'r=JSON.parse(STDIN.read); a=r["assets"].find{|x|x["name"].end_with?("_macOS_universal.pkg")}; abort("pkg 없음") unless a; puts a["browser_download_url"]') curl -fsSL "$URL" -o /tmp/gh.pkg && open /tmp/gh.pkg
  4. 4 Vercel CLI

    위 설치가 모두 끝난 후 터미널을 닫고 다시 여세요.

    npm i -g vercel

3단계. 로그인

GitHub 로그인 후 Git 설정과 Vercel 로그인까지 자동으로 합니다. 계속 터미널에서 실행합니다.

gh auth login git config --global user.name "$(gh api user --template '{{if .name}}{{.name}}{{else}}{{.login}}{{end}}')" EMAIL="$(gh api user/emails -q '.[] | select(.primary) | .email' | head -n1)" if [ -z "$EMAIL" ]; then EMAIL="$(gh api user -q .email)" fi if [ -z "$EMAIL" ]; then EMAIL="$(gh api user --template '{{.id}}+{{.login}}@users.noreply.github.com')" fi git config --global user.email "$EMAIL" vercel login
